When uploading pictures from a phone, its filename is going to be something cryptic like IMG_1234.JPG. Please provide your own filename. End the base name with "-1" to auto-increment. Use .jpg or .png as the extension.

If the uploaded file is a picture from a phone, it is going to be too big for your site. Sadly, resizing only works when uploading .jpeg, .heic and .png files. Feel free to specify a max width of 1200 pixels, for example.

If the filename you provided above ends in .jpg, you can specify a quality. Typically, a quality of 60 is not too bad and a quality of 90 is more than enough.

Finally, pick the files or photos to upload. Picture metadata is only removed if the pictures gets resized. Providing a new max width is recommended for all pictures.
